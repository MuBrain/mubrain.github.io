# Open Knowledge Format

L’Open Knowledge Format, ou OKF, est une manière de structurer la connaissance sous forme de fichiers Markdown enrichis par du frontmatter YAML.

Dans MuBrain, OKF sert de modèle d’inspiration pour organiser la connaissance de manière simple, durable et exploitable par des agents IA.

## Pourquoi cela compte pour MuBrain

MuBrain ne doit pas seulement être un site web. Il doit devenir un espace de connaissance structuré.

Cela signifie que les contenus doivent être :

- lisibles par des humains ;
- versionnés dans GitHub ;
- compréhensibles par des agents IA ;
- faciles à traduire ;
- séparés entre contenu public et connaissance de référence.

## Règle MuBrain

Le français canadien est la source de vérité.  
Les autres langues sont des traductions validées.
